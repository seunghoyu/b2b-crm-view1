B2B CRM 기능 명세서

구분 No Page 1depth 2depth 3depth 4depth 5depth 기능명 기능 설명 구현 방법
공통 1 전체 페이지 사이드 메뉴 - - - - 페이지 이동 및 활성 메뉴 표시 현재 페이지에 해당하는 메뉴로의 이동 및 활성 상태 표시. 각 페이지의 HTML 코드에 현재 메뉴에 해당하는 링크 부분의 배경색을 다르게 지정하여, 사용자가 어떤 메뉴를 보고 있는지 시각적으로 알려줍니다.
2 하위 메뉴 토글 '문자관리', '이메일 관리' 등 특정 메뉴의 하위 목록 표시/숨김 처리. 메뉴 제목 옆의 화살표 버튼을 클릭할 때마다, 아래에 숨겨져 있던 하위 메뉴 목록을 보여주거나 다시 숨깁니다.
3 외부 시스템 이동 안내 일부 메뉴 클릭 시 외부 시스템으로 이동함을 안내. '문자발송' 등 특정 하위 메뉴를 클릭하면, "최고운영자 페이지로 이동합니다" 라는 안내창을 먼저 보여줍니다.
4 상단 헤더 - - - - 페이지 제목 표시 각 페이지의 기능명과 부가 설명 표시. 각 페이지마다 제목과 설명이 HTML 코드에 직접 작성되어 있습니다.
화면 5 b2b_dashboard_tailwind.html 관리메뉴 상담 현황 - - - 상담 현황 대시보드 조회 등록된 상담 내역의 목록 조회. 페이지가 열릴 때, 미리 준비된 전체 상담 목록 데이터 중에서 최신 15개의 데이터만 골라 화면의 표에 자동으로 그려줍니다.
기능 6 필터 - - 기간/조건별 상담 내역 검색 다양한 조건(기간, 분류, 상태 등)으로 상담 내역 검색. (현재는 화면 디자인만 구현됨) 사용자가 '검색' 버튼을 누르면, 입력된 조건에 맞는 상담 내역만 찾아서 표를 다시 그려주는 기능이 추가될 예정입니다.
7 기업명 검색 - 기업명 검색 및 자동 완성 입력된 키워드에 맞는 기업명 목록 제안 및 선택 기능. 기업명 입력칸에 글자를 입력하면, 그 글자가 포함된 기업 목록이 아래에 자동으로 나타납니다. 목록에서 원하는 기업을 클릭하거나 키보드로 선택하면, 기업명과 기업코드가 자동으로 채워집니다.
8 상담 목록 - - 상담 상세 조회 페이지 이동 상담 목록의 특정 행 클릭 시 고객 상세 페이지로 이동. 표의 각 줄을 클릭하면, 해당 줄에 맞는 고객의 상세 정보 페이지로 화면이 넘어갑니다.
화면 9 company_list.html 관리메뉴 기업 목록 - - - 기업 목록 조회 등록된 모든 기업 정보 조회. 페이지가 열리면 먼저 10개의 기업 목록을 보여주고, 사용자가 '더보기' 버튼을 누를 때마다 다음 10개의 목록을 아래에 이어서 보여줍니다.
기능 10 신규 기업 등록 기능 실행 '신규 기업 등록' 또는 'Excel파일 일괄 등록' 기능 실행 안내. 각 버튼을 클릭하면, 해당 기능이 실행된다는 안내창을 보여줍니다.
화면 11 customer_list.html 관리메뉴 고객 목록 - - - 고객 목록 조회 등록된 모든 고객 정보 조회. 페이지가 열리면 먼저 10명의 고객 목록을 보여주고, 사용자가 '더보기' 버튼을 누를 때마다 다음 10명의 목록을 아래에 이어서 보여줍니다.
기능 12 신규 고객 등록 페이지 이동 '신규 고객 등록' 버튼 클릭 시 등록 페이지로 이동. '신규 고객 등록' 버튼을 클릭하면, 새로운 고객 정보를 입력할 수 있는 페이지로 화면이 넘어갑니다.
화면 13 customer_registration.html 관리메뉴 신규 고객 등록 - - - 신규 고객 등록 화면 표시 신규 상담 및 고객 정보를 입력하기 위한 전체 폼 화면 표시. HTML 코드로 다양한 정보(이름, 아이디 등)를 입력할 수 있는 빈칸들이 미리 만들어져 있습니다.
기능 14 신규 고객 정보 등록 기업명 검색 - 기업명 검색 및 자동 완성 입력된 키워드에 맞는 기업명 목록 제안 및 선택 기능. 기업명 입력칸에 글자를 입력하면, 그 글자가 포함된 기업 목록이 아래에 자동으로 나타납니다. 목록에서 기업을 선택하면 담당자 정보 등이 자동으로 채워집니다.
15 - 신규 기업명 입력 처리 검색 결과에 없는 기업명 입력 시 '신규'로 처리. 기업명 검색 시 목록에 없는 이름을 입력하고 엔터키를 누르면, 해당 이름 옆에 '신규' 표시가 붙으며 새로운 기업으로 등록할 수 있게 됩니다.
16 - - - 신규 고객 정보 등록 입력된 모든 정보를 저장하고 고객 목록 페이지로 이동. '등록하기' 버튼을 클릭하면 "등록되었습니다" 라는 안내창을 보여준 뒤, 고객 목록 페이지로 화면을 이동시킵니다.
화면 17 manual_list.html 관리메뉴 메뉴얼 관리 - - - 메뉴얼 목록 조회 등록된 모든 메뉴얼 정보 조회. 페이지가 열리면 먼저 10개의 메뉴얼 목록을 보여주고, '더보기' 버튼을 누를 때마다 다음 10개 목록을 이어서 보여줍니다.
기능 18 필터 - - 메뉴얼 검색 다양한 조건(기업명, 제목 등)으로 메뉴얼 검색. 사용자가 여러 조건(기업명, 제목 등)을 입력하고 '검색' 버튼을 누르면, 조건에 맞는 메뉴얼만 찾아서 아래 표에 다시 보여줍니다.
19 검색 조건 초기화 모든 검색 필터의 입력 값을 초기화. '초기화' 버튼을 누르면, 검색을 위해 입력했던 모든 조건들이 지워지고 전체 메뉴얼 목록을 다시 보여줍니다.
모달 20 메뉴얼 목록 - - 메뉴얼 상세 보기 테이블의 특정 행 클릭 시 해당 메뉴얼의 상세 내용을 담은 모달 표시. 메뉴얼 목록 표에서 특정 줄을 클릭하면, 해당 메뉴얼의 전체 내용(응대절차, 고객전달문구 등)을 보여주는 상세 정보창(모달)이 화면 위에 나타납니다.
기능 21 상세 보기 모달 - - 메뉴얼 수정/저장 상세 보기 모달 내에서 내용 수정 및 저장. 상세 정보창에서 '수정' 버튼을 누르면 내용을 편집할 수 있게 바뀌고, 수정 후 '저장' 버튼을 누르면 저장되었다는 안내창을 보여줍니다.
화면 22 manual_registration.html 관리메뉴 메뉴얼 관리 - - - 신규 메뉴얼 등록 화면 표시 새로운 메뉴얼을 등록하기 위한 입력 폼 화면 표시. HTML 코드로 메뉴얼 정보를 입력할 수 있는 여러 빈칸들이 미리 만들어져 있습니다.
기능 23 신규 메뉴얼 등록 입력된 모든 정보를 저장하고 메뉴얼 목록 페이지로 이동. '등록하기' 버튼을 누르면, 필수 항목들이 모두 입력되었는지 확인합니다. 비어있는 항목이 있으면 안내창을 보여주고, 모두 채워졌으면 등록 안내창을 보여준 뒤 메뉴얼 목록 페이지로 이동합니다.
화면 24 customer_view.html - - - - - 고객 상세 조회 화면 표시 특정 고객의 모든 정보(기본정보, 상담이력 등)를 종합적으로 표시. 페이지가 열릴 때, 해당 고객과 관련된 모든 정보(상담내용, 상담이력, 학습현황 등)를 가져와 각 항목에 맞게 화면을 채워줍니다.
기능 25 응대 내용 - - - - 응대 내용 수정/저장/취소 상단 고정 영역의 상담 내용 수정, 저장 또는 취소. '수정' 버튼을 누르면 비활성화 되어있던 입력칸들이 활성화되어 편집이 가능해집니다. '저장'을 누르면 저장 안내 후 다시 비활성화되고, '취소'를 누르면 수정 전의 원래 내용으로 되돌아갑니다.
모달 26 기능 메뉴 회원 정보 - - - 회원 정보 조회 및 수정 '회원 정보' 버튼 클릭 시 회원 정보를 담은 모달 표시 및 수정 기능 제공. '회원 정보' 버튼을 누르면, 해당 고객의 상세 정보(이메일, 주소 등)를 보여주는 정보창이 나타납니다. 이 창 안에서 '수정하기' 버튼을 눌러 내용을 직접 고칠 수 있습니다.
27 모달 - - 비밀번호 초기화 사용자 비밀번호를 아이디와 동일하게 초기화. 회원 정보창의 '비밀번호 초기화' 버튼을 누르면, 정말 초기화할 것인지 다시 한번 물어보는 확인창이 뜹니다. '확인'을 누르면 비밀번호가 초기화되었다는 안내창을 보여줍니다.
모달 28 배송지 정보 - - - 배송지 정보 조회 및 수정 '배송지 정보' 버튼 클릭 시 배송지 정보를 담은 모달 표시 및 수정 기능 제공. '배송지 정보' 버튼을 누르면, 해당 고객의 배송지 정보(받는사람, 주소 등)를 보여주는 정보창이 나타납니다. 이 창 안에서 '수정하기' 버튼을 눌러 내용을 직접 고칠 수 있습니다.
모달 29 이메일 전송 - - - 이메일 전송 방식 선택 '이메일 전송' 버튼 클릭 시 전송 방식을 선택하는 모달 표시. '이메일 전송' 버튼을 누르면, '웹메일'과 '패스메일' 중 어떤 방식으로 보낼지 선택할 수 있는 창이 나타납니다.
모달 30 문자/알림톡 전송 - - - 문자/알림톡 발송 모달 표시 '문자 전송' 또는 '알림톡 전송' 버튼 클릭 시 메시지 발송 모달 표시. '문자 전송' 또는 '알림톡 전송' 버튼을 누르면, 해당 고객의 전화번호가 자동으로 입력된 메시지 발송창이 나타납니다. 알림톡의 경우 내용 수정이 불가능하도록 설정됩니다.
31 모달 - - 템플릿 검색 및 선택 모달 내에서 템플릿을 검색하고 선택하여 내용을 자동으로 채움. 발송창 안에서 템플릿 제목을 검색하여 목록을 필터링할 수 있습니다. 원하는 템플릿을 클릭하면 그 내용이 메시지 입력칸에 자동으로 채워집니다.
모달 32 메뉴얼 조회 - - - 메뉴얼 조회 모달 표시 '메뉴얼 조회' 버튼 클릭 시 현재 고객의 소속 기업에 해당하는 메뉴얼 목록 모달 표시. '메뉴얼 조회' 버튼을 누르면, 현재 보고 있는 고객의 소속 회사에 해당하는 메뉴얼만 자동으로 골라서 목록을 보여주는 창이 나타납니다.
33 모달 - - 모달 내 메뉴얼 검색 모달 내에서 제목, 상황개요 등으로 메뉴얼을 추가 검색. 메뉴얼 조회창 안에서 제목이나 내용 등으로 다시 검색하여, 원하는 메뉴얼을 더 쉽게 찾을 수 있습니다.
기능 34 상담이력 - - - - 상담이력 상세 조회 상담이력 목록에서 특정 항목 클릭 시 우측에 상세 내용 표시. 왼쪽의 상담이력 목록에서 특정 내역을 클릭하면, 오른쪽에 해당 상담의 자세한 내용(답변 포함)이 나타나고, 선택한 내역은 배경색이 바뀌어 표시됩니다.
모달 35 학습관리 - - - - 학습 관리 기능 선택 모달 표시 학습관리 목록에서 특정 강의 클릭 시 관리 작업(시험 재응시 등)을 선택할 수 있는 모달 표시. 학습관리 목록에서 특정 강의를 클릭하면, 해당 강의에 대해 할 수 있는 작업('시험 재응시', '수강기간 연장' 등) 버튼들이 모여있는 창이 나타납니다.
36 기능 선택 모달 수강기간 연장 - - 수강기간 연장 모달 표시 '수강기간 연장' 버튼 클릭 시, 기간을 새로 지정하는 모달 표시. 학습 관리 창에서 '수강기간 연장' 버튼을 누르면, 기존 창은 사라지고 수강 시작일과 종료일을 새로 설정할 수 있는 전용 창이 나타납니다.
37 연장 모달 - 수강기간 연장 정보 저장 새로 입력된 기간 정보 저장. 수강기간 연장 창에서 날짜를 지정하고 '저장하기' 버튼을 누르면, 정말 저장할 것인지 다시 한번 물어본 뒤 저장되었다는 안내창을 보여주고 창을 닫습니다.
